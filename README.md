# sortari
Structuri de Date - Proiect Sortari

 În acest proiect, am implementat următorii 6 algoritmi de sortare:
 * BubbleSort
 * CountSort
 * MergeSort
 * InsertionSort
 * RadixSort (varianta LSD)
 * QuickSort (pivot ultimul element și pivot mediana din 3)
 
 Au fost comparați timpii de rulare pe mai multe teste cu numere naturale/reale între cei 6 algoritmi, dar și cu timpul de rulare al algoritmului de sortare nativ al limbajul C++.
